# 5-stage-current-starved-ring-oscillator
5-stage Current-Starved Ring Oscillator designed and simulated in Tanner EDA using 250 nm CMOS technology.
## Overview
This project implements a 5-stage Current-Starved Ring Oscillator using PMOS/NMOS current-mirror biasing to control the delay of inverter stages and tune the oscillation frequency.

## Features
- PMOS/NMOS current-mirror biasing
- Five current-starved inverter stages
- Voltage-controlled frequency tuning
- 250 nm CMOS implementation

## Results
- Control Voltage Range: 0.6 V – 1.8 V
- Frequency Range: 199 MHz – 1.156 GHz
- Tuning Ratio: 5.8×

## Tools Used
- Tanner EDA
